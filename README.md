# spectator

Spectator | Missing Person Locator App

## Overview

Spectator is a cutting-edge mobile application designed to assist in locating missing individuals swiftly and effectively. Harnessing the power of facial recognition technology, Spectator streamlines the search process, ensuring faster responses and increasing the chances of reuniting missing persons with their families. This cross-platform app offers a user-friendly interface and leverages advanced algorithms to provide accurate results.

## Key Features

### Functionalities

- **Facial Recognition:** Spectator uses facial recognition algorithms to identify missing persons based on uploaded images.

- **User Account Management:** Users can create and manage their accounts, making it easier to track their submissions and access app features.

- **Complaint Submission:** Submit detailed complaints about missing individuals, including images and relevant information.

- **Search and Compare Reports:** Search for missing individuals by name, and compare the results to uploaded images.

- **Emergency Contacts:** Quick access to vital contact numbers in case of emergencies.

- **Offline Functionality:** Spectator can be used in offline mode to access emergency contacts, ensuring uninterrupted assistance.

- **Security Measures:** Stringent security protocols, including CNIC verification, safeguard user data and app credibility.

### Additional Features

- **Cross-Platform Compatibility:** Spectator is available on both Android and iOS, providing accessibility to a broad user base.

- **Third-Party API Integration:** The app leverages the FacePlusPlus API for enhanced facial comparison.

- **Real-Time Database:** Firebase services, including Firestore and Authentication, ensure real-time data synchronization and secure data management.

- **User Notifications:** Display toast notifications to users for important updates and actions.

- **Image Selection:** Users can select images from their device gallery or capture new photos using the camera.

- **PIN Code Fields:** Secure authentication processes with PIN code fields.

- **External URL Launch:** Launch external URLs for making emergency contact calls or accessing additional resources.

- **Interactive Photo Viewing:** Enhance the user experience with interactive photo viewing using the photo_view package.

## Installation

To install Spectator on your device, follow these steps:

1. Clone the repository to your local machine.
2. Open the project in Android Studio or Visual Studio Code.
3. Connect your device and build the app.

## Usage

1. Register or log in to your Spectator account.
2. Submit detailed complaints about missing individuals.
3. Use the search functionality to locate missing persons by name.
4. Compare the results with uploaded images for potential matches.
5. Access emergency contact numbers for quick assistance.



Happy searching and helping bring missing persons home with Spectator!
